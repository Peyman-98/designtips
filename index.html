Mean Stack به مجموعه ای از تکنولوژی هایی گفته میشود که بر اساس Javascript ساخته شده و برای طراحی برنامه های تحت وب بکارمیرود. Mean مخفف MongoDB, ExpressJS, AngularJS و Node.js است.

Angular js یک فریمورک برای ساخت اپلیکیشن های پویا تحت وب است که به شما اجازه می دهد به راحتی کدهای Html خود را گسترش دهید.

Node.js یک پلتفرم منبع باز و سمت سرور برای ارتباط سریع و چندگانه برای شبکه است. این برنامه برپایه Javascript می باشد.
